# Mockups - Interfaz

Fecha: 2026-07-09

Este archivo reúne los mockups iniciales para las pantallas principales.

- Pantalla de ventas (optimizada para flujo de caja): [docs/mockups/ventas.svg](docs/mockups/ventas.svg)
- Panel de administración / Dashboard: [docs/mockups/admin.svg](docs/mockups/admin.svg)

Descripción breve de cada mockup:

- Ventas: búsqueda rápida, listado por categorías, grid de productos con botón grande "Agregar", panel derecho con carrito y botón prominente "Finalizar venta". Diseño pensado para uso en tablet y computadora con botones grandes.

- Admin: sidebar de navegación, KPIs en tarjetas (ventas, gastos), tabla de productos más vendidos y espacio para CRUD de usuarios/productos y configuración de categorías.

Formato: los mockups están en SVG vectorial para fácil edición.
